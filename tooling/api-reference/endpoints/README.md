# Endpoints

The available endpoints for unSkript:

* [Executions](executions.md)
  * List all executions
  * Get details of a specific execution
* [RunBooks](workflows.md)
  * List RunBooks
  * RunBook Details
  * Execute RunBook
* [Schedules](schedules.md)
  * List all Schedules
  * Get Specific Schedule Details
  * Create a Schedule
  * Delete a Schedule
  * Update a Schedule
